# Watch-Dogs-Universe-Interactive-Web-Project

Ce projet est un site web inspiré de l'univers du jeu **Watch Dogs**.

Il a été réalisé dans le cadre de mon apprentissage du développement web afin de pratiquer la création d'un site complet en utilisant **HTML, CSS et JavaScript**.

L'objectif est de développer mes compétences en front-end tout en travaillant sur un projet concret et motivant.

## Langages utilisées

* HTML
* CSS
* JavaScript

## Objectifs du projet

* Concevoir un site web structuré
* Pratiquer la mise en page et le design avec CSS
* Ajouter des éléments interactifs avec JavaScript

## Fonctionnalités

* Navigation entre différentes sections du site
* Mise en page responsive (adaptée à différents écrans)
* Éléments interactifs avec JavaScript
* Animations et styles personnalisés

## Ce que j'ai appris

Ce projet me permet notamment de travailler sur :

* la structuration d'un projet web
* la manipulation du DOM en JavaScript
* les bonnes pratiques de développement front-end
* l'utilisation de Git pour suivre l'évolution du projet
